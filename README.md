# OpenFOAM CFD Homework Cases

This repo contains all the OpenFOAM cases I developed during my CFD course.  
I’ve organized everything in a way that makes it easy to navigate, run, and understand what each case is about.

## 🧪 Cases Overview

### 1. One-Dimensional Advection-Diffusion
- Demonstrates scalar transport
- Focus on numerical diffusion and discretization schemes

### 2. Lid-Driven Cavity
- Classic benchmark problem
- Shows vortex formation and steady-state flow

### 3. Boundary Layer
- Simulation of velocity boundary layer development
- Useful for studying near-wall behavior

### 4. Backward Facing Step
- Flow separation and reattachment
- Mesh refinement plays a key role

### 5. Von Kármán Vortex Street (Re = 60)
- Unsteady flow past a cylinder
- Demonstrates vortex shedding and periodic behavior
